# 🎴 本地专家大脑 · 十大实操锦囊

> 10 个最典型、最能体现"本地专家大脑独有价值"的场景。
> 每个 = **什么时候用 + 一句口令 + 为什么只有这套大脑能做到**（差异化 = 普通 ChatGPT / 裸 Obsidian / 云 RAG 都做不到的那一刀）。
> 配套《本地专家大脑绿皮书》——锦囊给"何时抽哪一招"，绿皮书给"为什么+怎么做透"。

| # | 锦囊 | 触发场景 | 一句口令 | 差异化（只有本地大脑能做） |
|---|---|---|---|---|
| ① | **找回记得说过但搜不到的原话** | 记得某观点/原话，出处忘了、关键词想不起来 | `qmd query "那句话的大意" -c brain` | grep 要精确词、ChatGPT 没你的私货——只有向量检索能召回"意思像字面不像"的 raw 原文 |
| ② | **决策前问"当年为什么这么定的"** | 要做新决策，想看历史类似决策的**理由** | 「我以前关于 X 是怎么决策的、为什么」 | 决策的"为什么"是你的私有推理，任何外部 AI 都没有——这是企业大脑 vs 搜索引擎的命根 |
| ③ | **让 AI 用"你的味道"写东西** | 写文章/方案，要你的风格和观点不要通用腔 | 「基于我库里的风格和历史观点写 X」 | 越用越懂你：喂过你的思考，AI 学到你的判断文风。ChatGPT 永远是路人腔 |
| ④ | **别被纪要里放大的数字坑了** | 要引用财务/业务数字做决策或对外 | 「这数字是 facts 还是 raw，核过吗」 | 你的大脑有 facts/raw 可靠性分层；ChatGPT 自信复述错的，裸笔记不分层 |
| ⑤ | **动手前先查"是不是已经有了"** | 要建页/写脚本/做分析，怕重造轮子 | `qmd query "这件事/这个能力" -c brain` | 复用收敛 vs 重写发散；外部 AI 根本不知道你已有什么 |
| ⑥ | **一摞散材料一次深挖成报告/播客** | 几十个源要深度综合或出音频 | 「把 X 主题的源喂 NotebookLM 做深研/播客」 | QMD 给片段、wiki 给骨架，"对一摞源做长篇综合+音频"只有 NotebookLM |
| ⑦ | **省 token：别让 AI 全文读爆账单** | 用 API 计费时问要翻多文件的问题 | 「用 qmd 检索后回答，别全文读」 | 只返 3-5 段相关片段，实测省 ~96% token；grep 命中就全文吞 |
| ⑧ | **在任何项目里调你的大脑** | 在某个 coding 项目里需要库里的背景 | 用跨项目只读检索通道（任何目录可用） | 知识不锁在一个 App 里，任何项目的 agent 都能拉；笔记软件做不到 |
| ⑨ | **AI 说"搞定了"先别信** | agent 报告完成（装好/跑通/改完） | 「给我硬证据，别只说搞定了」+ 负证据换工具复验 | 刻进纪律的"自报告失实是头号红线"——AI 协作时代最值钱的肌肉 |
| ⑩ | **PII 业务：本地可问答、对外脱敏** | 客户/敏感数据要既能问答又不泄露 | 「本地问答 X 敏感数据，不外传」 | 本地可问答 + git 不带走；云 RAG / ChatGPT 都得先上传你的隐私 |

## 一句话记忆

> **①②找回 · ③复利 · ④⑨验真 · ⑤⑥⑦提效 · ⑧跨项目 · ⑩守隐私。**

## 三式检索速记（锦囊①⑤⑦ 的底层）

```bash
qmd query "完整问题" -c brain     # 混合(BM25+向量+重排)，最准，默认用它
qmd vsearch "模糊意思" -c brain    # 纯向量：意思像字面不像
qmd search "精确词" -c brain       # 纯BM25：版本号/专名/确切词
```

> 配套：详细原理见绿皮书卷三（六块肌肉）、卷五（场景→系统映射）、卷八（A1-A9 速查卡）。
